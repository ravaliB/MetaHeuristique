2 projets dont l'un traite le domaine discret, l'autre le domaine continu: 
Domaine discret : Sites
Domaine continu :  Essaim_Particulaire

Pour lancer ces 2 projets :
- ouvrir Eclipse
- cliquer sur File -> import -> Existing Projet into Workspace
      -> cocher sur archive File et selectionner les archives.zip des 2 projets.
- Executer.
